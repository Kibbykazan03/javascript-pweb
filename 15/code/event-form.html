Mempelajari Event Form

Event Form adalah event yang berhubungan dengan elemen form seperti input, select, textarea, dll. Beberapa contoh event form antara lain submit, focus, blur, dll. ada beberapa event yang biasanya terjadi pada form: -> submit: saat kita melakukan submit atau mengirim data pada form; -> reset: saat kita melakukan clear data pada form;
